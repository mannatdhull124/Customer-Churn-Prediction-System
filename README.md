# Customer-Churn-Prediction-System


## Project Overview

This project predicts telecom customer churn using Machine Learning techniques. The system analyzes customer service usage, account information, and billing details to determine whether a customer is likely to leave the company.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Models
* Hyperparameter Tuning
* ROC-AUC Evaluation
* Streamlit Web Application
* Real-Time Churn Prediction

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## Results

* Accuracy: 80.48%
* ROC-AUC Score: 0.84

## Deployment

The project is deployed using Streamlit for interactive customer churn prediction.

## Business Impact

This system helps telecom companies identify customers at risk of leaving and enables targeted retention strategies.
